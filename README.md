# dyslexic-bible

## Description:
Make bible.com dyslexic-friendly, by using the "Dyslexie" font and making the background off-white.

## Installation:
1. Install the [Dyslexie font](https://www.dyslexiefont.com) on your computer. It's free for personal use.
2. Install the [Tampermonkey](https://tampermonkey.net) browser plugin. Tampermonkey allows you to install scripts that run when you visit a specified website. This allows you to customize your experience when visiting the site. *Please use discretion when installing scripts, as they can be malicious. Only install scripts from sources you trust.*
3. Visit https://raw.githubusercontent.com/natematykiewicz/dyslexic-bible/master/dyslexic-bible.user.js and you will be prompted to install the script. You should see this screen:
<img width="1440" alt="installation" src="https://user-images.githubusercontent.com/5104186/37258302-b30053ba-2543-11e8-9e68-c69c68d60899.png">

4. Click Install. The window will close and install the script. (Updates to the script will be installed automatically.)
5. Visit https://www.bible.com/bible and enjoy the ease of reading.
